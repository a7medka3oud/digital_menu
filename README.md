# Smart Burger Menu

منيو ديجيتال مبني بـ Vue ويعمل محليًا بدون إعدادات معقدة.

## التشغيل

```powershell
npm start
```

ثم افتح:

```text
http://localhost:5173
```

## التعديل السريع

- من الواجهة: افتح زر `تعديل المنيو` لتغيير الاسم، الشعار، الصورة، الألوان، الأقسام، الأصناف والأسعار.
- من الملفات: عدل `src/menu-data.js` لتغيير البيانات الافتراضية.
- التغييرات من الواجهة تحفظ في المتصفح تلقائيًا ويمكن تصديرها أو استيرادها كملف JSON.
